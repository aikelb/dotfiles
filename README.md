Install chezmoi chezmoi and use the repo:

## Linux
```bash
$ sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply $GITHUB_USERNAME
```

## Windows
```bash
scoop install chezmoi
chezmoi init --apply $GITHUB_USERNAME
```

## macOS
```bash
brew install chezmoi
chezmoi init --apply $GITHUB_USERNAME
```
